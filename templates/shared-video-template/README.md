# Shared Video Template

共享模板只维护一套，不为每期视频复制代码。

这是共享的视觉和音频结构模板，具体书名、作者、文案、图片和音频由每期本地配置注入：

- 开场：黑屏碎片拼接、书单滚动、短黑场、结果页定格。
- 正文：结果页无跳变桥接，2-3 张 AI 氛围图慢推近和交叉淡入。
- 字幕：白色德意黑风格、黑色文字阴影、无底板。
- 书名和作者：进入正文后顶部常驻。
- 音频：`story` 旁白预设 + BGM 混音系数 `0.32`。

每期只替换本地 episode 下的书籍配置、脚本、图片和音频。最终渲染由项目代理工作流调用。

共享模板本身不包含具体书单、成片、BGM 或口播文件。
