# Book Automation

这是一个通过自然语言制作图书带货短视频的开源工作流。

你不需要先学习代码。直接告诉 Codex 你想做什么即可，例如：

- “初始化这个图书视频项目。”
- “推荐五本适合做情绪共鸣类视频的书。”
- “我想制作一本关于孤独和自我成长的书。”
- “我想制作某一本书的视频。”
- “把当前文案换成我提供的版本。”
- “这版视频不满意，请保留模板，替换图片和文案。”

首次使用时，系统会检查本地环境，并询问你是否启用微信读书。启用后，系统会引导你在本地配置 API Key；拒绝启用也不会阻断工作，可以通过公开资料或你指定的书继续制作。

标准协作顺序是：先确定书目，再确认一版文案；文案确认后生成氛围图片；你提供剪映生成的口播音频后，系统完成音画对齐、配乐和最终视频。

每次替换都会先生成并检查新方案，确认成功后再覆盖旧方案。个人书单、参考视频、生成图片和成片只保存在本机，不属于公开仓库内容。音乐、音效和口播素材只有在确认拥有发布权时才应纳入仓库。

项目采用 Apache-2.0。微信读书、HyperFrames、GSAP、FFmpeg、字体和媒体文件各自遵循自己的许可条款。

更详细的内部生产规则见 `AGENTS.md` 和 `docs/book-video-playbook.md`。
